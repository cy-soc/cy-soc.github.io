# cy-soc.github.io

Landing page for the **International Workshop on Cyber Social Threats (CySoc)**.

**Live site:** [https://cy-soc.github.io](https://cy-soc.github.io)

## Workshop Editions

- [CySoc 2026](https://cy-soc.github.io/2026/) — ICWSM 2026, Los Angeles, CA, USA
- [CySoc 2025](https://cy-soc.github.io/2025/) — ICWSM 2025, Copenhagen, Denmark
- [CySoc 2024](https://cy-soc.github.io/2024/) — ICWSM 2024, Buffalo, NY, USA
- [CySoc 2023](https://cy-soc.github.io/2023/) — ACM Web Conference 2023, Austin, TX, USA
- [CySoc 2022](https://cysoc2022.github.io/) — ICWSM 2022, Atlanta, GA, USA

## File Structure

```
index.html            # Main landing page
css/styles.css        # Custom styles
data/organizers.json  # Organizer data (name, affiliation, links)
js/organizers.js      # Renders organizers from JSON
```
